## 快速开始
启动步骤
1. 创建项目：`bee api web_01_router`
2. 进入 `cd .\web_01_router\` 目录
3. 更新包：`go mod tidy`
4. 生成接口路由：`bee generate routers`
5. 运行项目执行 `bee run`
6. 访问接口：http://localhost:8080/v1/user
