

## 2、MySQL / PostgreSQL（主流关系型数据库）
 - 特点：真实项目中最常用，支持复杂 SQL、事务、索引优化等。 
 - 适合：中大型系统、Web后端、企业服务。 
 - Go推荐库：`go get github.com/go-sql-driver/mysql` 
 - 🔸为什么第二学：能学到连接池、事务、SQL优化、ORM（如 gorm）等实战内容。
